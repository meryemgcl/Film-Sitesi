# Film Dünyası (Movie App)

Film Dünyası, kullanıcıların popüler ve vizyondaki filmleri keşfetmesini, kategorilerine göre filtrelemesini, detaylı film kartlarını incelemesini ve favori listeleri oluşturmasını sağlayan Django tabanlı dinamik bir web uygulamasıdır.

Bu proje, modern web mimarilerini, ilişkisel veritabanı yönetimini ve kullanıcı dostu responsive (mobil uyumlu) arayüz tasarımlarını uygulamak amacıyla geliştirilmiştir.

---

## 🚀 Öne Çıkan Özellikler

* **Dinamik Film Kataloğu:** Yönetici (Admin) paneli üzerinden eklenen tüm filmlerin; afişleri, kısa özetleri ve IMDB puanlarıyla birlikte ana sayfada modern kartlar halinde listelenmesi.
* **Kategori ve Tür Filtreleme:** Filmlerin aksiyon, dram, komedi, bilim kurgu gibi türlerine göre anlık olarak filtrelenebilmesi.
* **Gelişmiş Detay Sayfaları (Movie Detail):** Seçilen filmin üzerine tıklandığında açılan; fragman linki, uzun özet, vizyon tarihi, yönetmen ve oyuncu kadrosu gibi detaylı içeriklerin sunulduğu özel alt sayfalar.
* **Kullanıcı Kimlik Doğrulama (Auth Sistemi):** Güvenli kayıt olma (Register), giriş yapma (Login) ve çıkış (Logout) mekanizmaları ile şifreli oturum yönetimi.
* **Responsive Tasarım:** Bootstrap altyapısı sayesinde bilgisayar, tablet ve mobil cihaz ekranlarıyla tam uyumlu arayüz.

---

## 🛠️ Kullanılan Teknolojiler

* **Backend (Sunucu Katmanı):** Python & Django Web Framework
* **Frontend (Arayüz Katmanı):** HTML5, CSS3, Bootstrap
* **Veritabanı (Database):** SQLite (Geliştirme ortamı için optimize edilmiş ilişkisel veritabanı)
